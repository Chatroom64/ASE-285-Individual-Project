# Individual Project: Chat Room 64

---

### Problem
AI Chatbots such as ChatGPT are useful for many things, but there is not much in the way of personalization.

---

### Solution
An AI chatbot that focuses on its presentation; is easily personalized by the user.

---

Features and Requirements: 
1. Chat Feature
   - As a user, I should be able to enter a message.
   - As a user, I should be able to clearly see the message that the chatbot responds with.
2. Chat Storage
   - As a user, I should be able to store my previous chats.
   - As a user, I should be able to restore my stored chats at any time.
3. Chat Creation/Deletion
   - As a user, I should be able to create a new chat.
   - As a user, I should be able to delete an existing chat.
4. GUI Preferences
   - As a user, I should be able to choose from various color themes.
   - As a user, I should be able to generate a theme to use for the site using the chatbot itself.
   - As a user, I should be able to view and collapse the side menu containing my stored chat logs.
